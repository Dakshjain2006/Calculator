# 🔢 Quantum Calculator

A visually stunning cyberpunk-aesthetic calculator with animated glow effects and full keyboard support.

## 🚀 Live Demo
[Dakshjain.github.io/quantum-calculator](https://Dakshjain2006.github.io/quantum-calculator)

## ✨ Features
- Full arithmetic: `+`, `−`, `×`, `÷`, `%`
- Keyboard shortcut support
- Cyberpunk neon glow design with CSS animations
- Chained operations with expression display
- Responsive layout

## 🛠 Tech Stack
| Layer      | Technology         |
|------------|--------------------|
| Markup     | HTML5              |
| Styling    | CSS3, CSS Variables, CSS Animations |
| Logic      | Vanilla JavaScript |
| Fonts      | Google Fonts (Orbitron, Rajdhani) |

## 📁 Project Structure
```
quantum-calculator/
```
